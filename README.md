# BufferDump

A macro for dumping a formatted buffer in C or C++, great for debugging protocols, or investagting structure formats 

example output:
```
int main () {
    int buffer[40] = {1,2,3,4,5,6,7,'g','k','.'};
    buffer[36] = 'G';
    
    DUMP_BUFFER(buffer, sizeof(buffer));

    return 0;
}
```
